## 🎙️ Flagship — DreamStage

> **An AI producer for people who've never opened a DAW.**
> Record a vocal straight in the browser, and DreamStage hands back a fully produced, mixed and mastered song — the beat is generated *around your melody*, not the other way around.

- 🎛️ **Vocal-first, tiered beat generation** — ACE-Step neural synthesis / MusicGen / real-instrument DSP synthesis, all following the recorded vocal
- 🌿 **Producer Cuts** — unlimited, branchable creative sessions with taste learning that adapts to what you accept, favorite, branch, or skip
- 🧠 **AI producer coaching** built on the Claude API
- 🎚️ **Automatic mastering** to −14 LUFS, with MP3 / WAV export
- ✅ **Solo-built**, with green CI on GitHub Actions — the QA half of me insists
